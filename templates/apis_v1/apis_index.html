We Vote APIs, v1

A list of available APIs. See also our API design google document.

Method Description
Voter Basic Data
deviceIdGenerate Generate a device_id (store in cookie) that can be linked to a voter_id
voterAddressRetrieve Retrieve an address for a voter
voterAddressSave Save or update address for an existing voter
voterCount Count the number of voters in the database.
voterCreate Create an empty voter entry attached to a voter_device_id
voterRetrieve/?format=json Export raw voter data to JSON format
GeoLocation
Ballot Contest Data
voterBallotItemsRetrieve Retrieve a list of ballot items to display.
Candidates & Measures
candidatesRetrieve Retrieve a list of candidates competing for an office to display.
Positions (Support, Oppose, Info Only) or Stars
opposeCount Return the number of organizations the voter follows + the voter's friends that oppose the specified candidate or measure
voterStopOpposingSave Save or update one voter's lack of opposition for one candidate or measure
voterOpposingSave Save or update one voter's opposition for one candidate or measure
supportCount Return the number of organizations the voter follows + the voter's friends that support the specified candidate or measure
voterStopSupportingSave Save or update one voter's lack of support for one candidate or measure
voterSupportingSave Save or update one voter's support for one candidate or measure
Organizations & People
organizationCount Count the number of organizations in the database.
organizationFollow Store that the voter wants to follow this organization.
organizationFollowIgnore Store that the voter wants to ignore this organization.
organizationStopFollowing Store that the voter wants to stop following this organization.
organizationRetrieve Retrieve one organization based on unique identifier.
voterGuidesToFollowRetrieve Retrieve a list of voter guides that a voter might want to follow.
